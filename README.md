# ZoomButtons

Zoom Buttons es una caja de controles para facilitar el uso de Zoom durante las Videoconferencias.

Se proveen 3 botones para Zoom:
  - Alt+A: Mute/unmute
  - Alt+V: Start/stop
  - Alt+Y: Raise/lower hand

Y dos botones para controlar el audio del equipo:
  
  - Volumen Up
  - Volumen Down
